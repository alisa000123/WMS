## Features

The system is able to:
- Track items in stock
- Handle incoming stock (adding items)
- Handle outgoing stock (removing items)
- Show current warehouse state

---

## Levels of Complexity

### Tier 1 (Basic)

- Tracks quantities of items (bulk storage)

### Tier 2

- Implements FIFO (First-In, First-Out) batch handling
- Oldest items are removed first

### Tier 3

- Tracks individual items separately

---

## User Interface

The system provides:

- Display of current stock
- Item count
- Item information (depending on the tier)
- Controls for:
  - Adding items
  - Removing items
